# PROSE — Heartbeat #48

**Theme**: Light
**App**: Long-form reading tracker and book notes — library, session timer, highlights, discover, year-in-reading profile
**Elements**: 500
**Screens**: 6

## Palette
- BG: `#FAF6F0` — Pantone Cloud Dancer inspired warm off-white (2026 Color of the Year reference)
- Surface: `#FFFFFF` — clean white for cards
- Parchment: `#F3EDE3` — warm aged card background
- Text: `#1E1812` — warm near-black (not cool grey)
- Sienna: `#B85A2A` — burnt sienna / clementine primary — reads as ink on paper
- Sage: `#4E7A56` — secondary for "read" / completed states
- Peach: `#E8A882` — soft peach for personal notes left-border
- Sand: `#C9A87C` — warm sand for discovery / wish-list

## Research Sources
- Lapa Ninja 2025 (lapa.ninja): Fundable landing page — Piazzolla display serif + Open Runde rounded sans pairing; unconventional serif for a finance product that created warmth — applied to reading context where serif is natural
- Lapa Ninja 2025: Paperclip — Instrument Serif + Inter Serif pairing; confirmed editorial serif direction
- Land-book 2026: Clementine (soft orange-peach) + off-white trending in wellness/lifestyle brands; earth neutrals dominant
- NNGroup 2025: Pantone Cloud Dancer — "calming, versatile neutral blank canvas" named 2026 Color of the Year — directly referenced for BG choice
- NNGroup article on button states: pressed-state feedback within 100–150ms — informed action button styling decisions

## 3 Key Decisions
1. **Display serif throughout UI**: Screen titles, book titles, pull-quotes, the hero logotype all use a Georgia/Playfair-inspired serif. This is unusual for a mobile app — but reading is a serif context. The font choice signals "this app is for serious readers" and separates PROSE from the generic Inter-everything app template. Inspired by Fundable's Piazzolla pairing from Lapa Ninja.
2. **Quote vs. note typography distinction**: In the highlights screen, saved quotations from books render in serif italic (someone else's words); personal notes render in Inter (the user's own words). The font switch is the semantic signal — no label needed, no badge color required. A single typographic convention that scales across the whole feature.
3. **Colour-coded left-border semantic system**: Peach = personal note/thought. Sage = read/completed. Sienna = primary action. Sand = discovery/wish-list. Applied identically across book-detail notes, library book cards, highlights, trending list items. One four-signal system that works without explanatory labels.
